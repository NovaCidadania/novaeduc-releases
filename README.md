# novaeduc-releases

Artefatos publicos do NovaEduc para distribuicao multiplataforma.

## Conteudo

- `source/` - tarballs do codigo-fonte com checksum SHA256 para Flathub
- `android/` - APK e AAB publicados por versao
- `windows/` - instaladores e binarios Windows publicados por versao
- `linux/` - pacotes Linux publicados por versao
- `macos/` - builds macOS publicados por versao
- `metadata/` - manifesto Flatpak renderizado, screenshots e textos legais

## Politica de codigo-fonte

O repositorio principal do NovaEduc permanece privado por razoes comerciais.
Este repositorio publico existe para distribuir snapshots versionados do codigo
fonte usados na build do Flathub e para publicar artefatos de release
multiplataforma.
